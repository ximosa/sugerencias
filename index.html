Tu Script de Asistente de Artículos

Sigue estos pasos para integrar el asistente inteligente de Gemini directamente en los artículos de tu blog.

Paso 1: Sube los Archivos de la Aplicación

Primero, asegúrate de haber subido todos los archivos de esta aplicación (index.tsx, App.tsx, las carpetas components y services, etc.) a un directorio público en tu servidor o hosting.

Paso 2: Copia y Pega este Script

Pega el siguiente bloque de código al final de la plantilla de tus artículos (por ejemplo, en tu archivo slug.html o similar), justo antes de la etiqueta de cierre </body>. 

<!-- Punto de montaje para la App de Sugerencias Gemini -->
<div id="root"></div>

<!-- Script para cargar la aplicación desde tu servidor -->
<!-- ¡IMPORTANTE! Reemplaza la URL con la ruta real donde alojaste el script -->
<script type="module" src="https://URL-DE-TU-SERVIDOR/index.tsx"></script>

¿Cómo funciona?

  • Este script crea un div vacío con id="root" en tu página.
  • Luego, carga tu archivo index.tsx desde el servidor donde lo alojaste.
  • La aplicación de React se ejecutará, leerá el contenido de tu artículo dentro de <div id="page-wrapper"> y se mostrará dentro del div#root.
