Tu Script de Asistente de Artículos

¡Integrar el asistente de Gemini en tu blog es muy fácil! Simplemente copia el siguiente bloque de código y pégalo en tu web.

Paso 1: Prepara tu Artículo

Asegúrate de que el contenido principal de tus artículos esté envuelto en un `div` con el id `page-wrapper`. El widget necesita esto para leer el texto. 

<div id="page-wrapper"> ... tu artículo aquí ... </div>

Paso 2: Copia y Pega este Script

Pega el siguiente bloque de código al final de la plantilla de tus artículos, justo antes de la etiqueta de cierre </body>. 

<!-- Punto de montaje para la App de Sugerencias Gemini -->
<div id="root"></div>

<!-- Script de la aplicación de sugerencias -->
<script src="https://sugerencias-iota.vercel.app/widget.js" defer></script>

¿Cómo funciona?

El script widget.js es un archivo autocontenido que se genera automáticamente en cada despliegue. Simplemente buscará el div#root en tu página y renderizará la aplicación de React allí.
